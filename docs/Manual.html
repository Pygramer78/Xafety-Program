Manual for Xafety-Program

Getting started

First, this portable application is powered by WinPcap that you need to install. Click the link and install. Now, inside the bin folder, take the file called xafety.exe to System32.

All the commands

So, now that you installed WinPcap, you'll be able to use this application.
All that you need is the commands;
      xafety sniff to sniff other dispositives
      xafety ssid to see the ssid
      xafety scan scan for worms in the connection
